The use cases covered by this application are the login, ordering from a 
restaurant, purchasing and viewing tickets, and creating and managing activities
from the administrative side. The login allows the user to register by
entering their email and password. They can also login after entering a username
and password as a user or as an admin. For ordering from a restaurant, the user
can view the menu, select quantities for each item, request customization, 
enter payment information, and place their order.  For purchasing tickets, the
user can select a ticket quantity for each ticket type and enter their payment
information to complete their purchase. For viewing tickets, the user can view 
the tickets that they have purchased.  On the admin side, the user can manage 
and create new activities within the park.
